# Desenho Infantil

Trabalho desenvolvido para a disciplina de computação gráfica.


Após usar
```git clone <repository>```

Use 
```cd desenho-infantil```

Instale as bibliotecas pendentes
```pip install -r requirements.txt```

E execute
```python main.py```
